Fitur

Lengkap untuk ujian
yang benar-benar aman

Tiap fitur dibuat khusus untuk kebutuhan CBT sekolah — bukan aplikasi generik.

🔒
Screen Pinning
Kunci layar via startLockTask(). Siswa tidak bisa keluar ke app lain. Unpin otomatis setelah selesai.
  • Konfirmasi sebelum pin
  • Deteksi status realtime
🛡️
Monitor App Terlarang
Background service cek tiap 2 detik. 50+ package, 9 kategori: browser, AI, sosmed, VPN, remote access.
  • WakeLock + START_STICKY
  • VPN debounce 30 detik
📷
Kamera Live
Preview kamera depan di sudut layar selama ujian berlangsung. Anti titip jawaban.
  • CameraX — lifecycle-aware
  • Indikator ● REC berkedip
⚠️
Sistem Pelanggaran
8 trigger dicatat otomatis. Setelah 3× pelanggaran sesi dihentikan dengan countdown 5 detik.
  • Back, Recents, fokus hilang, split screen
  • Banner + badge realtime
🌐
WebView Aman
Whitelist 9 domain Google. Blokir copy/paste, klik kanan, dan download via JS injection.
  • Chrome mobile user-agent
  • Halaman error + retry
🔍
Integrity Check
Deteksi root, ADB, dev mode, emulator, dan VPN saat launch. Laporan ditampilkan ke guru.
  • 8 path su + Magisk/SuperSU
  • AppOpsManager check
