Fitur Lengkap

Semua yang dibutuhkan
untuk ujian yang adil

Dibangun khusus untuk lingkungan sekolah. Setiap fitur dirancang untuk mencegah kecurangan tanpa mengorbankan kemudahan penggunaan.

🔒
Screen Pinning
Kunci layar menggunakan startLockTask() Android. Siswa tidak bisa keluar ke app lain selama ujian.
  • Konfirmasi dialog sebelum pin
  • Deteksi status pin realtime
  • Unpin otomatis setelah ujian selesai
📷
Kamera Live Preview
Preview kamera depan live di sudut layar selama ujian. Bukti visual untuk anti titip jawaban.
  • CameraX API (stabil & modern)
  • Indikator ● REC berkedip
  • Otomatis tersembunyi jika kamera tidak ada
🛡️
Monitor App Terlarang
Background service cek tiap 2 detik apakah ada 50+ aplikasi terlarang yang dibuka.
  • 9 kategori: browser, AI, sosmed, VPN…
  • WakeLock agar service tidak mati
  • START_STICKY untuk restart otomatis
📡
Status Realtime
4 indikator diperbarui tiap 2.5 detik: koneksi, kunci layar, keamanan perangkat, dan izin monitor.
  • Hint teks: tunjuk syarat yang belum terpenuhi
  • Tombol ujian hanya aktif jika semua ✓
  • NetworkCallback untuk status WiFi
⚠️
Sistem Pelanggaran
Setiap tindakan mencurigakan tercatat. Setelah 3 pelanggaran, sesi otomatis dihentikan dan guru diberitahu.
  • 8 trigger: Back, Recents, fokus hilang, split screen…
  • Banner pelanggaran 4 detik
  • Countdown kick 5 detik dengan dialog
🌐
WebView Aman
WebView dikonfigurasi khusus: hanya whitelist 9 domain Google, blokir copy/paste/download.
  • User-agent Chrome mobile (compat Forms)
  • JS injection blokir klik kanan
  • Halaman error + tombol retry
🔍
Integrity Check
Saat launch, app memeriksa kondisi perangkat untuk memastikan lingkungan ujian yang aman.
  • Deteksi root (8 path + Magisk/SuperSU)
  • Cek ADB, developer mode, emulator
  • Deteksi VPN aktif
💾
Persistensi Sesi
Data sesi disimpan ke SharedPreferences. Tidak hilang meski app di-kill paksa atau HP restart.
  • URL ujian, timer, jumlah pelanggaran
  • Riwayat sesi terakhir
  • Alasan kick tersimpan
🔔
Notifikasi Selesai
Alarm sekali bunyi 2.5 detik dan vibrasi singkat saat ujian selesai. Ringkasan waktu + pelanggaran.
  • Auto-stop setelah 2.5 detik
  • Volume alarm diatur 75% max
  • Ikon notifikasi monochrome (semua HP)
Sistem Keamanan

Ancaman yang dideteksi,
dan cara menangkalnya

Setiap vektor kecurangan umum dalam ujian CBT sudah diperhitungkan dan ditangani.

Ancaman yang Dideteksi

📱
Aplikasi Terlarang
Browser, ChatGPT, WhatsApp, Telegram, screen recorder, kalkulator advanced — 50+ package dari 9 kategori.
🌐
VPN / Proxy
Deteksi koneksi VPN pada semua network adapter. Dilaporkan dengan debounce 30 detik.
🖥️
Remote Access
TeamViewer, AnyDesk, scrcpy, Vysor, Microsoft RDP, KDE Connect — semuanya dideteksi.
📸
Screenshot & Screen Record
FLAG_SECURE memblokir screenshot di semua activity. Deteksi app perekam layar via UsageStats.
⚡
Perangkat Root / Emulator
Cek su binary di 8 path, package Magisk/SuperSU, build.prop writeable, dan fingerprint emulator.

Cara Kerja

Alur ujian dari awal sampai akhir

Lima langkah sederhana. Guru setup, siswa kerjakan, sistem jaga integritasnya.

1
🚀
Buka Aplikasi
Splash screen jalankan integrity check — root, ADB, dev mode, VPN. Navigasi otomatis ke Setup.
2
🔒
Kunci Layar
Guru tap "Kunci Layar", Android tampilkan dialog konfirmasi Screen Pinning.
3
📷
Scan QR Token
Setelah 4 status ✓, siswa scan QR dari guru atau input link ujian manual.
4
📝
Kerjakan Ujian
WebView load Google Forms. Security service monitor terus di background.
5
✅
Selesai & Submit
Dialog recap waktu + pelanggaran. Alarm bunyi. Layar unlock otomatis.
